# 🌱 UrbanGreen – Community Gardening Network

UrbanGreen is a mobile application designed to connect urban gardeners through a digital platform where users can share plant growth, identify plants using AI-powered scanning, and exchange gardening resources through a barter system.

The goal of UrbanGreen is to encourage sustainable urban gardening and community knowledge sharing.

---

## 📱 Features

### 🌿 Community Garden Feed
Users can share their plant growth updates with the community.

• Post plant updates  
• View plant posts from other users  
• Real-time data powered by Firebase Firestore  

---

### 📷 Plant Recognition Scanner
UrbanGreen includes a plant scanning feature that allows users to capture plant images and identify plant species using AI-based recognition.

• Camera-based scanning  
• AI-powered plant identification  
• Instant plant name results  

---

### 🔁 Barter Marketplace
Gardeners can exchange seeds, tools, or plant cuttings with other users.

• List items for exchange  
• Browse available items  
• Community-based resource sharing  

---

## ⚙️ Technology Stack

UrbanGreen is built using modern cross-platform technologies.

**Frontend**
- Flutter (Dart)

**Backend**
- Firebase Firestore (Realtime database)

**AI Integration**
- PlantNet API for plant identification

**Other Packages**
- image_picker
- http
- permission_handler

---

## ☁️ Firebase Integration

UrbanGreen uses Firebase Firestore for storing and retrieving data in real time.
